      41              : // The ~uvm_sequence_library~ is a sequence that contains a list of registered
      42              : // sequence types. It can be configured to create and execute these sequences
      43              : // any number of times using one of several modes of operation, including a
      44              : // user-defined mode.
      45              : //
      46              : // When started (as any other sequence), the sequence library will randomly
      47              : // select and execute a sequence from its ~sequences~ queue. If in
      48              : // <UVM_SEQ_LIB_RAND> mode, its <select_rand> property is randomized and used
      49              : // as an index into ~sequences~.  When in <UVM_SEQ_LIB_RANDC> mode, the
      50              : // <select_randc> property is used. When in <UVM_SEQ_LIB_ITEM> mode, only
      51              : // sequence items of the ~REQ~ type are generated and executed--no sequences
      52              : // are executed. Finally, when in <UVM_SEQ_LIB_USER> mode, the
      53              : // <select_sequence> method is called to obtain the index for selecting the

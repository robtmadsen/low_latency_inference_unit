      37              : //
      38              : //------------------------------------------------------------------------------
      39              : // Base class for all task phases.
      40              : // It forks a call to <uvm_phase::exec_task()>
      41              : // for each component in the hierarchy.
      42              : //
      43              : // The completion of the task does not imply, nor is it required for, 
      44              : // the end of phase. Once the phase completes, any remaining forked 
      45              : // <uvm_phase::exec_task()> threads are forcibly and immediately killed.
      46              : //
      47              : // By default, the way for a task phase to extend over time is if there is
      48              : // at least one component that raises an objection.  
      49              : //| class my_comp extends uvm_component;
      50              : //|    task main_phase(uvm_phase phase);
      51              : //|       phase.raise_objection(this, "Applying stimulus")
      52              : //|       ...
      53              : //|       phase.drop_objection(this, "Applied enough stimulus")
      54              : //|    endtask
      55              : //| endclass
      56              : // 
      57              : //   
      58              : // There is however one scenario wherein time advances within a task-based phase
      59              : // without any objections to the phase being raised. If two (or more) phases 
      60              : // share a common successor, such as the <uvm_run_phase> and the 
      61              : // <uvm_post_shutdown_phase> sharing the <uvm_extract_phase> as a successor, 
      62              : // then phase advancement is delayed until all predecessors of the common 
      63              : // successor are ready to proceed.  Because of this, it is possible for time to 
      64              : // advance between <uvm_component::phase_started> and <uvm_component::phase_ended>
      65              : // of a task phase without any participants in the phase raising an objection.
